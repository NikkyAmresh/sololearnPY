What is Scoreboard.js?

Scoreboard.js is an easy to implement and use JavaScript module which adds an online scoreboard to your game.

The looks of the scoreboard is easy to customize and CSS syntax is used for customizing the different scoreboard components.

The module uses Firebase service as its realtime data storage.

Follow the next steps for an easy installation:
(click underscored step to open)

            14) Creating a scoreboard:
            


            To create the Scoreboard you need to first initialize it by running:
            


            scoreboard = new Scoreboard(options);
            

            (more about the options below)
            


            It is highly recommended that you run this command ONLY AFTER the HTML page and included scripts are fully loaded.
            

            You can either use <body onload='onLoadFunc()'> or window.onload=onLoadFunc or the jQuery alternative $(document).ready(function() {...});
            


            Make sure that the scoreboard variable is declared somewhere before and is accessible from all scopes where it is used (or just put var scoreboard = null; on the top of your script)
            


            15) Scoreboard Methods:
            


            I) showScoreBoard();
            

            Opens the Scoreboard dialog.
            

            Call this method when user clicked on button to show the scoreboard.
            


            Usage example: 'btn_show' is the id of the button which shows the scoreboard
            


            document.getElementById('#btn_show').onclick=function(){
            

            scoreboard.showScoreBoard();
            

            }
            


            II) submitNewScoreDialog(score);
            

            Opens submit new score dialog with given score value.
            

            Call this method once the game ended.
            


            Usage example: 'gameOver()' is the function being called on game over event
            


            function gameOver(){
            

            scoreboard.submitNewScoreDialog(score);
            

            }
            


            16) Options:
            


            The options are just like any other CSS element: just add to the option you want a pair structured as "cssAttribute": "cssValue"
            


            The options are later passed to $(element).css(option) which apply them.
