The Indian Penal Code of 1860(An Act to provide a general penal code in India), sub-divided into 23 chapters, comprises 511 sections. The Code starts with an introduction, provides explanations and exceptions used in it, and covers a wide range of offences.


The Indian Penal Code is the official criminal code of India. It is a comprehensive code intended to cover all substantive aspects of criminal law.

Various sections of the Indian Penal Code are controversial. They are challenged in courts claiming as against constitution of India. Also there is demand for abolition of some controversial IPC sections completely or partially.
