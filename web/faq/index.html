What's the difference between x++ and ++x / x-- and --x?

++x increments the value of x (x = x + 1) and then uses it.
x++ uses the value of x before increasing it.
-- decreases the value (x = x - 1).

Examples:
x=5; // x is 5
cout<<x++; // outputs 5, but x is now 6

x=5; // x is 5
cout<<++x; // x is now 6, outputs 6

What are challenges in SoloLearn? How do they work?

Challenges are initiated by challenging another user in the Challenge section using a certain weapon (language). Challenges consist of 5 relevant programming questions which has to be completed by both users within a limited amount of time. The user with the most correct answers wins. If both sides get equal amount of correct answers, the challenge ends in a draw.

After initiating and completing 5 questions of the challenge session, the user who initiated the challenge will have to wait until the other user comes online and accept the challenge. Users have the option to decline the challenge. If the other user does not respond to the challenge invitation, the challenge will simply expire.

Users gain XP for winning challenges, and lose XP for losing challenges. The base algorithm of the XP system in challenges is as such:
Lose XP = 32 - Win XP

Answer bonus may also be given to beginners.

What is the modulus operator (%)?

The modulus operator calculates the remainder of an integer division.
For example: 5 % 3 = 2 because 3 fits once into 5 and 5 - (3 * 1) = 2
Another example: 21 % 4 = 1; 21 - (4 * 5) = 1
And another: 15 % 5 = 0; 15 - (5 * 3) = 0

Why should I comment my code?

Use comments as a documentation for your code. They'll help you (and others) to understand what parts of the code does. This comes in handy when you re-read your code after several months or if others want to understand your code.

You can use comments also for debugging. Just comment out the faulty lines and try to run your program again. You can then remove the comment step by step until you find the error. Or maybe you need to enter a username and password to run your program, just comment out the login, so you can use your program without any credentials.

You'll find an overview of how to comment in different languages in the Wikipedia.

What's the difference between Java and JavaScript?

Java is a programming language running in the Java Virtual Machine (JVM), whereas JavaScript is a scripting language interpreted by a web browser. They only have the name in common as Java was very popular when JavaScript was created and the developers of JavaScript thought it would make their language equally popular when they put Java into its name.

What is the <span> tag?

Taken from W3Schools:
It's used to group inline-elements in a document. It provides no visual change by itself. It provides a way to add a hook to a part of a text or a part of a document.

You can use it to style parts of your document individually without using other, semantically not fitting tags. <span> is also often used for interaction with JavaScript or to insert images from an image sprite via CSS.

What is the difference between / and //?

They're both used for division, / results in a float division, whereas // results in an integer division.

Example (only Python3; in Python2 both would result in integers unless at least one operand is a float):

10 / 3 = 3.3333333333333335

10 // 3 = 3
