How to Draw CSS Shapes

  1. The CSS Code snippet below each shape is editable.You can play around with sizes and colours to see the effect on shape.

  2. You can copy paste the snippets from output itself and use them in your codes.

  3. Numbers in [ ] in description indicate the shape number that was used to construct the shape.

21.Rhombus

Rhombus shape can be created by simplying rotating the square[1].
Other Method:Join Triangle Up[9] and Triangle Down[10].






22.Heart I

The Heart shape is created by joining two semi-circle up[5] on the top of a rhombus[21].






23.Heart II

The Heart shape is created by joining two semi-circle up[5] on the top of a Triangle Down[10].







24.Drop

The Drop shape is created by joining a Triangle Up[9] on the top of a Circle[3].







25.Star I

This Star shape is created by joining a Triangle up [9] and a Triangle Down[10].







26.Star II

This star shape is created by joining three rectangles[2] each rotated at a particular angle.







27.Star III

This Star shape is created by three Triangle down[10] rotated at a particular angle.










28.Diamond

The Diamond shape is created by joining a Trapezium[13] on the top of a Triangle Down[10].









29.Pentagon

The Pentagon shape is created by joining a Triangle Up[9] on top a Trapezium[13].








30.Hexagon

The Hexagon shape is created by joining two Trapezium[13] one of which is inverted.








31.Cloud

The Cloud shape is created by joining two Circles[3] on the top of a Capsule[15].








32.Moon

The Moon shape is created by making the circle[3] transparent and using box shadow of that circle.
